# Flight Search Automation Suite

## Prerequisites
- Java 11+
- Maven 3.6+
- Chrome Browser
- ChromeDriver (in PATH)

## How to Run

1. Clone the repo:
   git clone https://github.com/yourusername/FlightSearchTest.git

2. Navigate:
   cd FlightSearchTest

3. Run Test Suite:
   mvn test

## Output
- Screenshots saved as:
  - flight_search.png
  - iframe_popup.png (if iframe found)

## Notes
- This suite uses Selenium WebDriver + TestNG
- Implicit and explicit waits are used
- JavaScriptExecutor handles scrolling
- Dynamic dropdowns, checkboxes, calendar are tested
